# SEO Tools

A free, professional web application providing essential SEO utilities for web developers, content creators, and digital marketers.

## Tools

| Tool | Description |
|------|-------------|
| **Meta Tag Analyzer** | Analyze any URL's meta tags, Open Graph data, headings, and get actionable SEO recommendations |
| **Meta Tag Generator** | Generate optimized HTML meta tags including title, description, keywords, and social tags |
| **Keyword Density Checker** | Analyze keyword frequency and density for single words and bigrams |
| **Word Counter** | Count words, characters, sentences, paragraphs, and estimate reading time |
| **Open Graph Generator** | Create Open Graph meta tags for social media previews |
| **Robots.txt Generator** | Build a valid robots.txt file to control search engine crawling |
| **Sitemap XML Generator** | Generate a properly formatted XML sitemap |
| **URL Slug Generator** | Convert text into clean, SEO-friendly URL slugs |

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```

The app will be available at `http://localhost:5000`.

## Running Tests

```bash
pip install pytest
python -m pytest tests/ -v
```

## Tech Stack

- **Backend:** Python / Flask
- **Frontend:** HTML, CSS, JavaScript
- **Parsing:** BeautifulSoup, lxml
